## Constraints
- **技術**：必須維持純 Laravel + Blade + SSR；不得引入 SPA / Nuxt / React / Vue / Inertia / 前端 routing
- **套件可使用**：`laravel/socialite`、`mcamara/laravel-localization`、`spatie/laravel-translatable`（其餘需理由）
- **安全**：
  - OAuth 來源新建帳號**絕不**繼承 `is_admin`
  - 任何登入流程都必須通過 age-gate（目前以 cookie 控制；若改成 DB 欄位需獨立 migration）
  - 既有密碼重設流程的 token 與 throttling 不可弱化
- **SEO**：每新頁面備齊 unique title / meta description / 主 H1 / canonical / OG / hreflang；分頁/過濾頁要考慮 noindex 或 canonical 防 cannibalization
- **部署**：保留單容器；不引入 Redis、MySQL、Postgres、Elasticsearch、訊息佇列服務
- **內容審核**：所有使用者輸入欄位（暱稱、房名、棋格文字、卡牌、輪盤）需走 moderation
- **DB**：SQLite 為主；schema 變更需新增 migration（不可改既有檔），需可 rollback
- **中介層註冊順序**：locale middleware 必須在 `EnsureAdmin` 與 `age.verify`（alias 註冊於 `bootstrap/app.php`，指向 `app/Http/Middleware/AgeVerification.php`）**之前**完成註冊，避免後台/年齡頁載入錯誤語系

### 4. 主題 C — i18n
- **4.1** 套件選型比較表：`mcamara/laravel-localization` vs `spatie/laravel-translatable` vs Laravel 內建 `Lang` —— 推薦 + 理由
- **4.2 URL 策略 — 必含完整對照表（含舊 URL 向後相容規則）：**

  | URL prefix | App locale | hreflang | 中文標籤 |
  |---|---|---|---|
  | /tw | zh_TW | zh-TW | 繁體中文 |
  | /cn | zh_CN | zh-CN | 簡體中文 |
  | /jp | ja    | ja    | 日本語   |
  | /en | en    | en    | English |

  **舊 URL 過渡規則（鎖定如下，實作者必須照做）：**
  - 根網域 `/` → Accept-Language 偵測 + cookie → 301 重導至 `/{locale}`
  - 其他既存非 prefix URL（`/login`、`/register`、`/privacy`、`/terms`、`/age-verify`、`/games/*`、`/play/*`、`/admin/*`、`/boards/*` 等）→ **一律 301 永久重導至 `/{default_or_cookie_locale}/{path}`**，保留 query string 與 fragment，HTTP status 必須是 301
  - admin 與 boards 等 auth-only 路由仍在 prefix 之下（例如 `/tw/admin`、`/tw/boards`），locale prefix 不影響 EnsureAdmin / auth middleware 判定
  - canonical 標籤永遠指向自己的 locale 版本，不指向非 prefix 的舊 URL
  - sitemap.xml 只列 prefix 版本，不列舊 URL
  - 舊 URL **不發 noindex**，靠 301 自然交棒給新 URL
- **4.3** 靜態翻譯檔結構：`lang/{locale}/auth.php` / `ui.php` / `seo.php` / `legal.php` / `games.php` / `play.php` / `errors.php`
  - 完整列出第一階段需翻譯的 lang key 清單（每檔至少列出 5–10 個 key 樣本）
- **4.4** 動態 DB 翻譯欄位設計

  - 兩種選項比較：JSON 欄位（`content_translations: {zh_TW:…, ja:…}`） vs spatie translatable trait
  - 推薦方案 + migration 草案 + Eloquent accessor 用法
- **4.5** 後台翻譯編輯介面

  - UI：locale 切換 tab + 欄位 inline 編輯
  - fallback 規則：缺翻譯時顯示 zh-TW；admin 可標記「待翻譯」狀態

- **4.7** 預設語系與 fallback 策略
